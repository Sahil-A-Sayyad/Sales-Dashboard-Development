# Sales-Dashboard-Development(Freelance)

# Project Overview
Developed an interactive Power BI sales dashboard to deliver actionable insights into overall, divisional, dealer‑level, and monthly sales performance. The project combines Power Query for data transformation and DAX for advanced calculations, enabling clear trend analysis and year‑over‑year comparisons.

# Tools & Technologies
- Power BI – Dashboard design & visualization
- Power Query – Data cleaning & transformation
- DAX – Advanced measures & time intelligence

# Dashboard Pages
- Overview
- KPI cards for quick performance snapshots
- Stacked column & line charts for sales and volume comparisons
- Monthly & yearly slicers with calculated measures for YoY growth percentage
- Division‑wise Report
- Regional breakdowns (FYMTD vs LYMTD, FQTD vs LQTD, FYTD vs LYTD)
- Yearly stacked column charts for comparative insights
- Dealer View Report
- Dealer‑level insights with pie charts
- Conditional formatting (red/green indicators for YoY performance)
- Interactive slicers for dynamic filtering
- TOPN parameter to display top dealers by FYTD sales
- Monthly Sales Report
- Comparative analysis of FMTD vs LMTD
- Monthly sales trends and performance tracking

# Business Impact
- Enabled year‑over‑year performance tracking
- Delivered regional storytelling with division‑level insights
- Provided dealer‑level monitoring with dynamic ranking
- Highlighted monthly sales trends for short‑term evaluation
